# Bargain_Information

## サイト概要
スーパーや薬局側の安売りセール等の情報を宣伝する

### サイトテーマ
SNS

### テーマを選んだ理由
最近一人暮らしを初めて、近くにスーパーが数件ある中でどこの何が安いのかを知れたらと思った。
また、店側からしたら紙媒体で宣伝するよりSNSを通した方が広告費も安く済むため

### ターゲットユーザ
・スーパー
・薬局
・自炊をする人（主婦、一人暮らし）

### 主な利用シーン
・スーパー、薬局　→　安売り、タイムセール、イベントを宣伝したいとき
・自炊をする人（主婦、一人暮らし）　→　買い物に行く時
